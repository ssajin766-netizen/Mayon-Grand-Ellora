# 🏢 Mayon Grand Ellora
## Smart Apartment & Community Management System

Mayon Grand Ellora is a modern Apartment and Community Management System designed to simplify the daily operations of residential communities.

The application provides an all-in-one digital platform where residents and administrators can manage maintenance bills, complaints, notices, emergency contacts, visitor information, online payments, and much more.

# ⚙ Installation

Clone Repository

```bash
git clone https://github.com/ssajin766-netizen/Mayon-Grand-Ellora.git
```

Move inside project

```bash
cd Mayon-Grand-Ellora
```

Install Packages

```bash
npm install
```

Create .env

```env
MONGO_URI=

SESSION_SECRET=

SECRET_KEY=

EMAIL_USER=

EMAIL_PASS=

RAZORPAY_KEY_ID=

RAZORPAY_KEY_SECRET=
```

Start Server

```bash
node server.js
```

Server

```
http://localhost:3000
```
